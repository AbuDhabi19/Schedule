# Time Management Application

This is a Windows Presentation Foundation (WPF) application for managing time and tasks during a semester. It allows users to add, delete, and track modules for a semester while calculating self-study hours based on the number of credits, class hours, and weeks in the semester.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- **Visual Studio**: You'll need a version of Visual Studio (e.g., Visual Studio Community) installed on your computer. If you don't have it, you can download it from the official Visual Studio website (https://visualstudio.microsoft.com/).

## Installation

1. Clone or download this repository to your local machine.

```bash
git clone https://github.com/your-username/time-management-app.git


Open the project in Visual Studio.

Build the solution by clicking the "Build" button or using the "Build" > "Build Solution" menu option.

Usage
Run the application by clicking the "Start Debugging" button (green arrow) or pressing F5.

The application window will appear, allowing you to perform the following actions:

Add Module: Fill in the module details (Module Code, Module Name, Credits, Class Hours per Week, Number of Weeks) and click the "Add Module" button.
Delete Module: Select a module from the list and click the "Delete Module" button to remove it.
Reset Values: Click the "Reset Values" button to clear all input fields and the modules list.
Semester Parameters: Enter the number of weeks in the semester and the start date.
The modules list displays the added modules, along with calculated self-study hours.

To close the application, click the "X" button in the top-right corner of the window.
